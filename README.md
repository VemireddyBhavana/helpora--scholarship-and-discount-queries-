
# Helpora — Scholarship and Discount Queries

An AI-powered campus fee-support agent built as part of the NIAT Agentic AI Workshop.

## Overview
Helpora helps students with scholarship, discount, and refund-related support questions by:
- checking scholarship eligibility
- checking invoice records
- checking payment records
- creating follow-up tasks for human review when needed

## Main use case
A student, Riya (S-7-043), reports that her scholarship has not been applied to her fee invoice. Helpora checks her eligibility and invoice records, identifies the issue, and creates a task for manual review if required.

## Features
- Scholarship eligibility lookup
- Invoice verification
- Payment record lookup
- Refund / duplicate-payment support
- Human task creation for follow-up cases

- Python
- SQLite
